📌 🛡️ RWG SZ (Random Word Generator - SeriZ)

A lightweight, fast, and secure tool for generating high-entropy random strings.

This project is designed to enhance the security of subscription links and inbound paths in traffic management panels such as Sanaei, Marzban, and other Xray/V2Ray-based panels.

---

✏️ 🧐 Why RWG SZ?

In server management panels, a common security vulnerability is the predictability of inbound paths or subscription URLs. Bots and unauthorized actors often use brute-force techniques to discover these links.

The Solution: Use long, complex, and unpredictable strings.
The higher the entropy, the harder it is to guess. RWG SZ allows you to generate these strings in seconds with professional-grade customization.

---

✏️ ✨ Key Features
• 🚀 Zero Installation: Runs directly in your web browser (Standalone HTML).
• 🔐 Privacy First: All generation happens locally on your device. No data is ever sent to a server.
• ⚙️ Fully Customizable:
• Adjust exact string length.
• Toggle numeric characters (1-9).
• Add custom branding/fixed text at the Start, Middle, or End of the string.
• 📋 One-Click Copy: Fast Copy-to-Clipboard functionality with visual feedback.
• 📱 Modern UI: Responsive design, optimized for both Mobile and Desktop users.

---

✏️ 🛠 How to Use

1. Clone this repository or simply download the index.html file.
2. Open index.html in any modern web browser.
3. Configure your desired length and branding text.
4. Click Generate Secure String.
5. Copy the result and use it in your panel's Subscription Path or Inbound Path settings.

📚 ✅ Technical Recommendations
For maximum security in V2Ray/Xray environments:
• Length: Set between 45 to 70 characters.
• Branding: Placing your custom text in the Middle of the string makes pattern recognition significantly harder for scanners.
